# Ostendo Wishlist

Future features and enhancement ideas. Contributions welcome!

## 1. ASCII Art Animations

Moving, spinning, or bouncing ASCII art elements on slides. Think DVD screensaver bounce, spinning logos, or animated text effects.

## 2. Slide Transition Animations

Fade, slide-left, dissolve, or other transition effects between slides instead of instant switching.

## 3. Runtime Element Editing

Edit slide content (bullets, code blocks, titles) directly during the presentation without switching to the markdown file.

## 4. Movable/Resizable Presenter Notes Buffer

Allow the speaker notes panel to be repositioned (top, bottom, side) and resized dynamically during presentation.

## 5. Expanded Exec Language Support

- C/C++ compilation templates (auto-detect gcc/clang)
- Auto-compiler detection for compiled languages
- Code injection/preamble support (e.g., auto-import common modules)

## 6. Presentation Centering Options

Vertical and horizontal centering modes for slide content, with configurable alignment per slide or globally.

## 7. Mermaid Graph Rendering

Render Mermaid diagrams (flowcharts, sequence diagrams, state machines) inline as ASCII art or protocol images.

## 8. Export to HTML/PPTX/PDF

Export presentations to portable formats for sharing with non-terminal users.

## 9. Configurable Author Footer

Per-presentation author name, date, or custom text in the status bar or footer area.

## 10. Status Bar Position

Option to place the status bar at the bottom of the screen instead of the top.

## 11. Better Themes

- Gradient backgrounds
- Highlighted/decorated title bars
- Theme variants (light/dark pairs)
- Custom accent colors via front matter override
