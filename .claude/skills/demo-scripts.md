# Demo Scripts

Guide for live code demos in Ostendo.

## Modes

- `+exec`: run code, capture output
- `+pty`: run in pseudo-terminal, keep ANSI codes

## Languages

Python, Bash, JavaScript, Ruby, Rust, C, C++, Go.

## Limits

64KB code, 1MB output, 30s timeout, setsid sandbox.

## Tips

- Ctrl+E to run
- Add labels with curly-brace label syntax
- Use preamble directives for shared setup
- Keep demos under 5 seconds
